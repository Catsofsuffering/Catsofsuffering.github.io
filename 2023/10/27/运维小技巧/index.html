使用Python命令

使用 python -m http.server 8080 命令可以快速启动一个简单的 HTTP 服务器，能够在本地共享文件。这对于快速共享文件或在本地测试网页非常有用。

  2. 将文件放入要共享的目录：

  3. 通过浏览器访问文件：

打开浏览器并输入 http://localhost:8080 或 http://127.0.0.1:8080，然后浏览服务器根目录中的文件并下载它们。
